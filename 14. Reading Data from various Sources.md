# Reading Data from various Sources

## 1. CSV (Comma-Separated Values)

- Most common format.

- Python (pandas):

```python
import pandas as pd

df = pd.read_csv("data.csv")
```

---

## 2. Excel Files (.xls, .xlsx)

- Useful for business datasets.

- Python (pandas):

```python
  df = pd.read_excel("data.xlsx", sheet_name="Sheet1")
```
